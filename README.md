# MemoryStack AI

Memory-aware AI system with semantic retrieval, contextual memory injection, and long-term memory summarization using PostgreSQL, pgvector, and Cohere embeddings.

## Features

- Semantic memory storage
- Vector similarity retrieval
- Retrieval transparency
- Context injection visualization
- Long-term memory summarization
- Persistent AI memory architecture

## Tech Stack

- Python
- PostgreSQL
- pgvector
- Cohere Embeddings
- HTML/CSS/JavaScript

## Architecture

User Query
↓
Embedding Generation
↓
Semantic Retrieval
↓
Memory Filtering
↓
Memory Summarization
↓
Context Injection
↓
LLM Response
